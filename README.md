# 🎮 AULauncher

A modern, lightweight mod launcher for Among Us that allows you to easily download and switch between different mod presets.

## ✨ Features

- **📚 Preset Library** - Browse and download mod presets easily
- **⚡ One-Click Launch** - Download, install, and launch mods with a single click
- **💾 Persistent Settings** - Choose your Among Us installation path

## 🚀 Download

### Latest Release
Download the latest version from the [Releases page](https://github.com/mehzxzz/aulauncher/releases)

**System Requirements:**
- Windows 10/11
- Among Us installed (Only steam is supported currently)
- .NET 8.0 Runtime (Compiled with the launcher)

## 📖 Usage

### First Time Setup

1. **Download** `AULauncher.exe` from the Releases page
2. **Run** the application (no installation required)
3. **Navigate to Settings** (⚙️ icon in sidebar)
4. **Select your Among Us folder** (the one containing `Among Us.exe`)
5. **Click "Save Path"**

### Loading a Preset

1. **Go to Library** (📚 icon in sidebar)
2. **Browse available presets**
3. **Click "Play"** on any preset
4. **Enter password** if required
5. **Wait for download and installation**
6. **Game launches automatically**

### Filtering Presets

Use the filter buttons at the top of the Library page:
- **All** - Shows all available presets
- **🔒 Locked** - Shows only password-protected presets
- **🔓 Unlocked** - Shows only open presets (recommended)
